# Installation
```sh
$ git clone git@github.com:igor-anferov/smart-guide.git
$ cd smart-guide
$ npm install -g yarn
$ yarn install
```
# Running dev-server
```sh
$ yarn start
```
### Available endpoints
| URL                     | Description       |
| ------                  | ------            |
| http://0.0.0.0:3000/    | Application       |
| http://0.0.0.0:3000/api | API documentation |
